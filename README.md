# CarbonSenseAI


## Overview
CarbonSense AI was built at the NVIDIA Agents for Impact Hackathon (1st place out of 100+ teams). It analyzes AI workloads and infrastructure choices to estimate environmental impact and suggest more sustainable alternatives.

## Tech Stack
- Python
- NVIDIA Nemotron LLM
- LangChain / LangGraph
- Prompt Engineering

## Features
- Estimates carbon footprint of AI workloads based on GPU usage
- Converts impact into real-world equivalents (e.g., flights, trees)
- Suggests more sustainable alternatives
- Generates structured sustainability reports
